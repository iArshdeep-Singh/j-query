Traverse up the DOM tree to find ancestors of an element

A descendant is a child, grandchild, great-grandchild, and so on

children() method returns all direct children of the selected element. This method only traverses a single level down the DOM tree

children("tag") method is used with parameter to filter the search for children

find("tag") method returns descendant elements of the selected element, all the way down to the last descendant

find("*") returns all descendants
